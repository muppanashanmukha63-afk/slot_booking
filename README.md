# Experience Slots Web App
An interactive and lightweight web application that demonstrates slot-based UI concepts and dynamic user interaction. This project highlights reusable components and responsive design using core web technologies.
## Live Demo
experiences-slots.netlify.app
## Features
- Interactive slot-style interface  
- Dynamic content rendering  
- Reusable UI structure  
- Fast and lightweight  
- Fully responsive (mobile and desktop)
Tech Stack
- react.js
- node.js
- mongoDB
- CSS
